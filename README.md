# 💱 实时汇率换算器

一个简洁现代的实时汇率换算器，支持10种主要货币的实时换算。

## ✨ 功能特点

- **实时汇率**: 自动获取最新汇率数据
- **10种货币**: CNY, USD, EUR, GBP, JPY, HKD, AUD, CAD, SGD, KRW
- **简洁UI**: 现代化设计，响应式布局
- **智能换算**: 输入金额自动计算
- **货币交换**: 一键交换源货币和目标货币
- **键盘快捷键**: 支持快速操作
- **离线备用**: API失败时使用预设汇率

## 🚀 快速开始

### 方法1: 使用Node.js服务器运行

```bash
# 进入项目目录
cd ~/.openclaw/workspace/currency-converter

# 启动服务器
node server.js
```

然后在浏览器中打开：http://localhost:3000

### 方法2: 直接打开HTML文件

直接双击 `index.html` 文件在浏览器中打开（需要联网获取汇率数据）。

## 🎯 使用方法

1. **输入金额**: 在"从"输入框中输入要换算的金额
2. **选择货币**: 选择源货币和目标货币
3. **自动计算**: 系统会自动显示换算结果
4. **交换货币**: 点击⇅按钮交换货币
5. **重置**: 点击重置按钮恢复默认设置

## ⌨️ 键盘快捷键

- **Enter**: 在金额输入框按回车进行换算
- **Ctrl+R**: 刷新汇率数据
- **Ctrl+S**: 交换货币
- **Esc**: 重置所有设置

## 🔧 技术实现

### 前端技术
- **HTML5**: 语义化结构
- **CSS3**: Flexbox布局，渐变背景，动画效果
- **JavaScript (ES6+)**: 面向对象编程，异步操作
- **Fetch API**: 获取汇率数据

### 汇率API
- 主要API: ExchangeRate-API (https://api.exchangerate-api.com/v4/latest/CNY)
- 备用数据: 预设的汇率数据（当API不可用时使用）

## 🔄 汇率更新机制

1. **首次加载**: 页面加载时自动获取最新汇率
2. **定时更新**: 每5分钟自动更新一次汇率
3. **手动更新**: 按Ctrl+R或刷新页面手动更新
4. **错误处理**: API失败时使用预设汇率并显示提示

## 🔧 自定义配置

如需添加更多货币，修改 `index.html` 中的 `<select>` 元素和 `app.js` 中的预设汇率数据。

## 🐛 故障排除

1. **无法获取汇率**: 检查网络连接，或等待API恢复
2. **页面空白**: 检查浏览器控制台错误信息
3. **计算结果错误**: 刷新页面获取最新汇率
